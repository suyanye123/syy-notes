# 基础

#### python安装依赖

将项目中的依赖包导入到一个文本文件中

```powershell
pip freeze > requirePackage.txt
```

批量安装依赖项

```powershell
pip install -r requirePackage.txt
```

#### 常用指令

```bash
su root	//进入管理员
exit	//退出root

apt-get update	//更新软件包列表
apt-get upgrade	//更新软件
sudo apt-get clean	//清理旧软件缓存
```
